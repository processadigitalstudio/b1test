## Cómo publicar en GitHub Pages (gratis)

1. Ve a https://github.com y crea una cuenta si no tienes
2. Crea un repositorio nuevo (ej: `examen-b1`)
3. Sube los dos archivos (index.html y admin.html)
4. Ve a Settings → Pages → Source: "main" → Save
5. Tu link será: `https://TU_USUARIO.github.io/examen-b1/`
6. Estudiantes entran a ese link
7. Admin entra a: `https://TU_USUARIO.github.io/examen-b1/admin.html`

## Para audios (ElevenLabs gratis)
1. Ve a https://elevenlabs.io (plan gratis = 10,000 caracteres/mes)
2. Pega el script del Audio 1 o Audio 2
3. Descarga el .mp3
4. Entra al admin → sube el archivo directamente desde la interfaz

## Notas importantes
- Los resultados se guardan en el navegador (localStorage)
- Si los estudiantes presentan desde computadores DISTINTOS, los resultados quedan en cada computador
- Para centralizar resultados desde distintos dispositivos → conectar a Firebase (pídeme y lo hacemos)
- El examen bloquea copy/paste y clic derecho
- Cada audio se puede escuchar máximo 2 veces
- El temporizador es de 55 minutos automático
